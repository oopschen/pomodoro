# Pomodoro tools  
This is a easy use, command line based tool for the good of pomodoro.  
  
## Feature  
* timer for work  
* timer for break
* cusimized notify sound
  
## Usage  
  
    pomodoro [-h]  [-w 25] [-r 5] [-n /home/ray/notify.sound]
